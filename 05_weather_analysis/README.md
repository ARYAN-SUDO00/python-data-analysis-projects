# Weather Data Analysis(Australia)

## Description
This project analyzes Australian weather data from 2008-2025 using Python and Pandas to understand temperature trends, rainfall patterns, and regional climate differences.

## Dataset
- Name: Australian Weather Dataset
- Source: https://rattle.togaware.com/weatherAUS.csv
- File Used: weatherAUS.csv

## Tasks Performed
- Calculated the average minimum and maximum temperatures
- Identified the hottest and coldest months based on average temperatures
- Analyzed average monthly rainfall
- Determined the month with the highest total rainfall
- Identified the hottest city/region based on average maximum temperature
- Found the wettest city/region based on total rainfall

## Tools Used
- Python
- Pandas
