# Netflix Content Analysis (2008-2021)

## Description
A Python-based analysis of Netflix's library using Pandas to uncover content trends, growth patterns, and library highlights from 2008 to 2021.

## Dataset
- **Name:** Netflix Movies and TV Shows (Kaggle) by Shivam Bansal
- **Source:** [Kaggle Link](https://www.kaggle.com/datasets/shivamb/netflix-shows)
- **File Used:** `netflix_titles.csv`

## Tasks Performed
- Counted and compared the total volume of Movies vs. TV Shows.
- Identified the most frequently occurring genres across the platform.
- Extracted years from date strings to list content added annually from 2008â€“2021.
- Located the oldest and newest titles based on release years and upload dates.
- Converted string-based date columns into proper `datetime` objects for accurate sorting.
