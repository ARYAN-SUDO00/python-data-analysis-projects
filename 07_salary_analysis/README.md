# Employee Salary Analysis

## Description

This project analyzes employee salary data using Python and Pandas to understand salary distribution, department trends, and payroll insights.

## Dataset

* **File Used:** `employee_salary.csv`
* **Type:** Structured CSV dataset containing employee information such as department, job title, gender, and yearly salary.

## Tasks Performed

* Calculated average salary by department
* Identified highest paying job and department
* Identified lowest paying job and department
* Calculated average salary by gender
* Computed total yearly payroll expense

## Tools Used

* Python
* Pandas
