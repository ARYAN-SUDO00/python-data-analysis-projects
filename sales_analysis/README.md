# Sales Data Analysis

## Description
This project analyzes the Superstore Sales Dataset using Python and Pandas to understand sales performance, revenue trends, and ordering patterns.

## Dataset
- **Name:** Superstore Sales Dataset  
- **Source:** Kaggle (by Rohit Sahoo)  
- **Link:** https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting  
- **File Used:** train.csv  

## Tasks Performed
- Calculated total revenue
- Analyzed sales distribution across product categories
- Identified the top 5 selling products
- Determined the month with the highest number of orders and total sales revenue

## Tools Used
- Python
- Pandas
